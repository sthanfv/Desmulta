# Guía de Identidad Visual — Paleta de Colores Desmulta

Este documento detalla los códigos de color oficiales utilizados en la plataforma para asegurar la consistencia de marca y la calidad visual (MANDATO-FILTRO).

---

## 🎨 Color Primario (Premium Gold)
El alma de la interfaz. Un dorado vibrante que proyecta autoridad y éxito.

| Atributo | Valor (HSL) | Valor (HEX) | Uso Principal |
| :--- | :--- | :--- | :--- |
| **Primary** | `45, 100%, 52%` | `#FFBF00` | Botones de acción, iconos clave, acentos destacados. |
| **Primary Shadow**| `45, 100%, 20%` | `#664D00` | Sombras proyectadas, estados activos profundos. |

---

## 🌓 Modo Claro (Monet Light)
Inspirado en interfaces limpias con tintes cálidos de "oro sutil".

| Variable | Valor (HSL) | Valor (HEX) | Descripción |
| :--- | :--- | :--- | :--- |
| **Background** | `45, 30%, 98%` | `#FCFAF2` | Fondo principal con ligero tinte crema. |
| **Foreground** | `222.2, 84%, 4.9%` | `#020817` | Texto principal (Azul Oxford profundo). |
| **Card** | `0, 0%, 100%` | `#FFFFFF` | Superficies de tarjetas y contenedores. |
| **Muted** | `45, 20%, 96%` | `#F9F7F0` | Fondos de inputs y elementos secundarios. |
| **Border** | `45, 20%, 90%` | `#EBE6D9` | Líneas de división y bordes sutiles. |

---

## 🌑 Modo Oscuro (Monet Dark - AMOLED)
Diseñado para el máximo contraste y ahorro de energía en pantallas OLED.

| Variable | Valor (HSL) | Valor (HEX) | Descripción |
| :--- | :--- | :--- | :--- |
| **Background** | `0, 0%, 0%` | `#000000` | Negro puro (AMOLED Black). |
| **Foreground** | `0, 0%, 98%` | `#FAFAFA` | Texto principal (Blanco Ártico). |
| **Card** | `0, 0%, 2%` | `#050505` | Tarjetas con ligero relieve sobre el negro puro. |
| **Secondary** | `0, 0%, 10%` | `#1A1A1A` | Elementos de UI secundarios. |
| **Border** | `0, 0%, 15%` | `#262626` | Delimitadores sutiles en entorno oscuro. |
